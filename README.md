# motor-mapeamento-tendencias-

Mapeador de tendências em tempo real para identificação de necessidades de mercado e desenvolvimento de soluções multiplataforma.

## 🚀 Objetivo do Projeto
Simplificar a dor do nosso cliente final através de um super-app multiplataforma. O motor foi desenhado para entender o algoritmo de busca do usuário (via integrações de APIs do Google e dispositivos móveis) para centralizar e facilitar serviços de:
* 💳 Pagamentos integrados e facilitados.
* 📅 Reservas em tempo real.
* 🏛️ Acesso a Serviços Públicos desbancarizados/desburocratizados.
* 🗺️ Mapeamento de rotas inteligentes e rápidas.

## 🛠️ Tecnologias Utilizadas
* **Python:** Inteligência do motor de busca, análise de dados e algoritmos de recomendação.
* **PostgreSQL:** Banco de dados relacional com suporte a geolocalização para armazenar usuários, logs de buscas, rotas e transações.
* **Power BI:** Dashboards estratégicos para análise de volumetria de dores e conversão de soluções.

## 🗄️ Estrutura do Banco de Dados (schema.sql)
O banco foi modelado para garantir alta performance e cruzamento de dados inteligente:
1. `utilizadores`: Cadastro e nível de perfil do cliente.
2. `historico_buscas`: Onde o algoritmo Python salva as tendências capturadas em tempo real.
3. `servicos_disponiveis`: Catálogo de soluções, reservas e serviços públicos parceiros.
4. `pagamentos`: Registro financeiro unificado de todas as classes sociais.
5. `rotas_frequentes`: Mapeamento de trajetos rápidos e alertas de trânsito.
